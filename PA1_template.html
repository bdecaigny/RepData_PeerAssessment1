Imputing missing values

As mentioned earlier, these calculations use the cleaned up data set, i.e. with the NA rows removed. In the original table, there are 2304 NA rows on a total of 17568 rows. In stead of working with reduced data set as with did thusfar, we could also choose to replace the missing values with calculated data, e.g. the average number of steps for that time interval, or for that date. As we see quite a strong pattern in the average steps per interval, we prefer to replace the NA values with the interval means.

Let’s check what the impact is on the total number of steps per day resp. on the average steps per day.

The mean and average of steps taken per day can also be recalculated:

##    Min. 1st Qu.  Median    Mean 3rd Qu.    Max. 
##      41    9820   10800   10800   12800   21200

We can conclude that replacing the NAs with interval means has an impact on the quantile boundaries, but not on the mean or the median. When we recreate a plot for the average number of plots per interval, we see there is no impact either.

This was expected, of course, as we replaced the missing values with the averages already used in the plot.
